# Ecommerce Rewired

- This ecommerce project has been developed in order to demonstrate how AR can be used across several domains ranging from Education, Real Estate, Fashion and even Machinery.
- As ecommerce can cover several domains or categories, so we chose to develop an ecommerce application that can actually help to show the potential growths that AR can provide.
- Next, we used flutter as these applications have been developed that can be used in the mobile as a device for the same.
- Also, upon studying the best platform to develop the mobile application with minimal overhead for the setup, we chose flutter over React Native, as flutter offers numerous advantages over React Native.

## NOTE : Ecommerce application has been made using flutter. The augmented reality part has been made via Unity3D. The relevant images for the same has been attached below, which have been demonstrated using the apk.
